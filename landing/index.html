Get started

Install in under a minute.

Once it's on the Chrome Web Store this is one click. Until then, you can load it manually today.

1

Download

Grab the extension .zip and unzip it.

2

Load it

Open chrome://extensions, turn on Developer mode, click Load unpacked, pick the extension folder.

3

Trade

Open a TradingView chart, join a room (or a Public ticker room), allow your mic — you're on call.

⚠️ "Add to Chrome" goes live with the Chrome Web Store listing. Google only allows one-click install from the Web Store, so for now use the manual steps above. Once published, the button here will install it instantly.
